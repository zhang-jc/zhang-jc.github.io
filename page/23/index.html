Shell 脚本报错：“line 10: [: =: unary operator expected”。根据提示信息找到报错的程序是：

if [ $OPERATION == "scp" ]; then

报错原因是变量 OPERATION 的值为空，那么程序就变成了：

if [ == "scp" ]; then
