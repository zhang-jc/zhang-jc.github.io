其实在 CentOS 7.4 下已经存在 libffi.so.6，解决方法是创建软连接，如下：

# cd /usr/lib64
# ln -s libffi.so.6 libffi.so.5
# ln -s libffi.so.6.0.1 libffi.so.5.0.1
