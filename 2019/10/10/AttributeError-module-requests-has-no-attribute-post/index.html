AttributeError: module 'requests' has no attribute 'post'

发表于 2019-10-10 更新于 2023-10-17 分类于 工具箱 ， Requests 阅读次数： Changyan：

发现是因为安装的 Requests 模块权限的问题。安装完成后模型权限如下：

# ls -l ./lib/python3.7/site-packages/requests
