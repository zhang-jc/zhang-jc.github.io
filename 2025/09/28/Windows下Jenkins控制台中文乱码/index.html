环境信息

  • Windows 11 家庭中文版
  • java 21.0.4 2024-07-16 LTS
  • Jenkins 2.452.3

解决方法

增加系统 JAVA_TOOL_OPTIONS，并设置值为 -Dfile.encoding=GBK。

打开设置方法：桌面上右键点击“此电脑”图标，选择“属性”。
